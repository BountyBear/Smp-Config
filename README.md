# Smp Essentials

## How To Use
### Command
* * /smp [Pearls/Shields/Pots/Tier2Pots/Single Mace/EnchantLimit] [On/OFF]

# Dependency
* Skript 2.9.5
* Skbee 3.6.3
* skreflect 2.5.1

## Features
* Single Mace
* Disable all pots
* Disable teir 2 pots
* Disable Shields
* Disable Pearls

* PVP

- - Protection
- - Sharpness
